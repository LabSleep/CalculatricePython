
## Description

Une calculatrice en ligne de commande qui permet d'effectuer les 4 opérations de base :

- Addition
- Soustraction
- Multiplication
- Division

## Comment l'utiliser

1. Lance le programme avec Python :
```bash
   python calculatrice.py
```
2. Entre le premier nombre
3. Entre le deuxième nombre
4. Choisis une opération (1, 2, 3 ou 4)
5. Le résultat s'affiche !
